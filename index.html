Visualization 1: Interactive Building Distribution

Description

This interactive dot plot compares the number of state-owned buildings across Illinois counties. Each circle represents a county, positioned by building count (x-axis) and sorted vertically for easy comparison.

Design Choices

  • Encodings: Quantitative counts (x), nominal counties (y)
  • Interactivity: Multi-select circles to highlight counties
  • Color: Highlighted selections use categorical scheme

Data Processing

Filtered null counties and aggregated counts using Altair's count() transform.

Visualization 2: Size vs. Year Analysis

Description

Scatter plot showing building square footage versus construction year, with county filtering to analyze regional trends.

Design Choices

  • Encodings: Years (x), square footage (y), county (color)
  • Interactivity: Dropdown filter + zoom/pan
  • Scales: Non-zero baselines for proper trend visibility

Data Processing

Cleaned numeric fields and added "All" option to dropdown programmatically.
